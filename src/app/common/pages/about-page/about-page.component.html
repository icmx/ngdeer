ngdeer — это неофициальный веб-клиент для проекта «Подслушано». С помощью ngdeer можно читать секреты так же, как с официального сайта или из приложений — только без рекламы и других отвлекающих элементов.

Сейчас ngdeer поддерживает только чтение: здесь нет регистрации или авторизации, поэтому отправить свой секрет через ngdeer не получится. Возможно, эти функции появятся в будущем.

Оригинальные истории созданы и принадлежат авторам проекта «Подслушано» (ideer.ru).

Исходный код ngdeer доступен под лицензией MIT (github.com/icmx/ngdeer).
